\uhhhh где hhhh - шестнадцатеричный код 

Подъязык символов
[ ] - любой символ из перечисленных
[^ ] - любой символ кроме перечисленных
^ - если не идёт сразу за скобкой, считается простым символом
- (минус) - если идёт первым или последним - считается простым символом
a-z (минус) - если идёт в середине, то считается "от a до z"

Модификаторы
{n} - n повторений
{n,} - n и более
{n,m} - от n до m 
? == {0,1}
+ == {1,}
* == {0,}
? - модификатор модификатора "нежадный"

^ - начало строки
$ - конец строки

Комбинаторы
| - "или" имеет наименьший приоритет

Группировка
( ) - скобки с захватом
(?: ) - скобки без захвата
(?= ) - заглядывание вперёд, если следует
(?! ) - заглядывание вперёд, если не следует
